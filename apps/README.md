# WAIBA App UI Projects

This folder contains app-specific UI learning and app-navigation projects.

Each app gets its own project:

- `apps\vscode`
- `apps\antigravity`
- `apps\<app-slug>`

Use the dedicated app-navigation shared library at:

`C:\git\wb_pr\_shared\apps\app-navigation`

This app-navigation layer is separate from web navigation. The first phase is always observation only: create a first impression, classify the visible UI, and model dynamic workspace regions before executing actions.
